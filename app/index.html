Serious about Security

Fully encrypted and securely stored

We adhere to industry-leading standards and best practices to manage our network, servers, web and client applications, and set security policies across our organization.

We employ SSL exclusively for all communication between client applications and servers. Data is protected in transit and storage with industry standard cryptographic protocol using 256 bit encryption keys.

Applications are subject to strict quality testing and security review.

We take all necessary measures to control access to sensitive data, application data, and cryptographic keys.

Two-factor authentication and strong password controls are required for administrative access to systems.

Networks are strictly segregated according to security concerns. Modern, restrictive firewalls protect all connections between networks.
